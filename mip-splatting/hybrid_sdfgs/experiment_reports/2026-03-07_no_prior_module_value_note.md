# 2026-03-07 结论记录：无 Prior 模块的价值（Normal-Gated + Adaptive Tangential Expansion）

## 模块定义
本模块不依赖外部图像 prior，仅使用局部几何信息（锚点中心/法向）进行 pseudo-SDF 融合增强：
1. `法向一致性门控`：抑制法向冲突邻居的影响。
2. `自适应切向扩张`：根据局部法向一致性调节切向作用范围，法向方向不盲目扩张。

---

## 在本轮实验中的定位
- 这是一个 **no-prior robustness module**，目标不是“全场景无脑提分”，而是：
- 在锚点稀疏、法向噪声、局部 outlier 这些困难条件下，降低 pseudo-SDF 的结构性误差。

### 结果 1：复杂形状稀疏场景
- baseline_mae_global: `0.182608`
- gated_mae_global: `0.131962`
- baseline_mae_sparse: `0.106220`
- gated_mae_sparse: `0.065089`

解读：全局与稀疏区都显著下降，说明该模块在真实“缺覆盖”区域有效。

### 结果 2：两高斯参数扫（distance × condition）
- 噪声 sweep：正收益格子 `45/70`
- outlier sweep：正收益格子 `60/70`

进一步观察：
- 当两高斯中心距离较大（例如 sep >= 44°）时，收益更稳定。
- 当存在明显 outlier（单点法向偏差 >= 10°）时，收益几乎全域稳定为正。
- 在小间距 + 低噪声下，收益可能不稳定或略降（方法偏鲁棒性增强，而非简单场景提分器）。

---

## 模块价值结论（可用于论文叙述）
1. **不依赖外部 prior**：避免了外部生成模型带来的风格偏移和视角不一致风险。
2. **困难区间收益明确**：在稀疏/噪声/outlier 条件下，误差下降稳定且可复现。
3. **机制可解释**：通过“法向门控 + 自适应扩张”直接作用于局部几何融合，符合几何直觉。
4. **适合作为基础模块**：可作为后续更复杂方法（如 densify-to-SDF、多阶段优化）的底座，而不是和 prior 强耦合。

---

## 风险与边界
- 不是全条件单调增益；在低难度条件可能收益有限。
- 门控阈值和扩张系数需按数据分布调参。
- 本轮仍是 2D toy 验证，后续需迁移到 3D 主线做同类稳健性验证。

---

## 建议的后续验证
1. 在 3D 主线做 `with/without` 模块对照（固定其余训练流程）。
2. 指标分层报告：全局误差 + 稀疏区域误差 + 视角一致性误差。
3. 固化默认参数区间（先用 sweep 中稳定区间作为初值）。
